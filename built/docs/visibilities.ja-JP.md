# 投稿の公開範囲

投稿には以下の公開範囲を指定することができます

## 公開
全てのユーザーが見ることができます  
公開タイムライン (ローカル/グローバル) に表示されます

## ホーム
全てのユーザーが見ることができます  
公開タイムライン (ローカル/グローバル) には表示されません

## フォロワー
自分のフォロワーのみ見ることができます

## ダイレクト
指定したユーザー (メンションまたはユーザーを追加したユーザー) のみ見ることができます

## 公開 (ローカルのみ)
全てのユーザーが見ることができますがリモートへは公開されません  
公開タイムライン (ローカル/グローバル) に表示されます

## ホーム (ローカルのみ)
全てのユーザーが見ることができますがリモートへは公開されません  
公開タイムライン (ローカル/グローバル) には表示されません

## フォロワー (ローカルのみ)
自分のローカルフォロワーのみ見ることができます
